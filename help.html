Aqua Morning Checks - Maintenance Logs System Documentation

1. Authentication & Security

The system operates in Read Only mode by default to ensure data integrity. To make any changes, you must unlock the interface.

  1. Enter your Login ID (your name or initials) in the yellow field.
  2. Enter the Admin Code.
  3. Click Unlock.

Note: All actions performed while unlocked are logged with your Login ID and a timestamp.

2. Visual Indicators (The Legend)

The table uses color-coding to communicate the status of tasks at a glance.

White / Checked		 Standard task completion. Toggle with Left-Click.
Red Background		 Indicates an issue, failure, or required rerun. Toggle with Right-Click.

3. Updating the Master Template

To change the list of tasks globally, you must upload a new CSV file.

  • Column A: Task Description (URLs will automatically become clickable links).
  • Column B: Scheduled Time (e.g., 5:25am).
  • Columns C-G: Leave blank for checkboxes, or type N/A to disable a specific day.

Once your file is ready, click Update Template while in Edit Mode.
